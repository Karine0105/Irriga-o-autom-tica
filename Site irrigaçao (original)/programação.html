Processo de programação

Nosso projeto de irrigação automática foi desenvolvido com o objetivo de modernizar o processo de cuidado com as plantações. Criamos um aplicativo com uma interface intuitiva e programação baseada em blocos, permitindo monitorar e controlar a irrigação de forma automática. O sistema mede o nível de umidade do solo em tempo real, indicando precisamente a quantidade de água que a planta necessita. Quando o nível de umidade cai para 25%, uma ordem automática é enviada para iniciar a irrigação, garantindo que as plantas recebam água na medida certa, sem desperdício. Esse sistema automatizado facilita o cuidado com as plantações e otimiza o uso de recursos hídricos. Além disso, desenvolvemos um site utilizando HTML e CSS para apresentar todas as informações do projeto. A página foi planejada para ser organizada e visualmente atraente, explicando o funcionamento do sistema, suas vantagens e aplicações práticas. Com uma interface clara, o site facilita o acesso aos detalhes técnicos e inovações do projeto. A automação de sistemas de irrigação também é viabilizada com o uso de Arduino, utilizando programação em C++. Esse projeto oferece uma solução prática e eficiente para manter as plantas bem cuidadas, sem a necessidade de intervenção manual constante. O código C++ para a irrigação automática segue uma estrutura que inclui a definição dos pinos, a configuração inicial e o loop principal, assegurando o funcionamento adequado do sistema.
